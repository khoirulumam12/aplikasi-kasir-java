# Aplikasi Kasir Cafe (Java)

Aplikasi ini dibuat untuk membantu manajemen transaksi di toko kelontong.

## Fitur Utama
- Transaksi Penjualan & Cetak Struk
- Laporan Harian & Bulanan
- CRUD Data Barang

## Teknologi yang Digunakan
- Bahasa: Java (NetBeans IDE)
- Database: MySQL
- Library: JasperReport (untuk laporan)

## Cara Menjalankan
1. Import file `database.sql` ke PHPMyAdmin.
2. Buka project di NetBeans.
3. Atur koneksi database di file `Koneksi.java`.
4. Run file `Login.java`.# aplikasi-kasir-java
Aplikasi Point of Sale (POS) berbasis desktop menggunakan Java Swing dan MySQL.
